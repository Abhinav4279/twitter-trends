# twitter-trends

> API to get last 24 hrs top twitter trends in India.


## Installation
`git clone https://github.com/Abhinav4279/twitter-trends`\
`cd twitter-trends`\
`npm install`

## Running Development Server
`npm start`

## Usage
`curl http://localhost:3000/`

## Example
#### Request
`curl http://localhost:3000/`
